# scaler_ctrl 说明文档

## 文件位置
`C:\Users\huawei\Desktop\project_codex\rtl\ctrl\scaler_ctrl.sv`

## 作用
作为整条图像缩放流水线的控制器，串接读源图、行缓存、缩放核心、行输出缓存和写回模块。

## 主要职责
- 锁存一次缩放任务配置
- 根据缩放核心的 `line_req_y` 决定是否需要装载新源行
- 协调何时启动第一行/下一行输出
- 统计已启动和已写回的目标行数

## 工作流程
1. `start` 时保存地址、stride 和图像尺寸参数。
2. 运行态中根据 `line_req_y` 发起 DDR 读和行缓存装载。
3. 启动缩放核心输出一行像素。
4. 行输出缓存准备好后发起写回。
5. 全部目标行写完后进入 done；任一子模块报错则进入 error。

## 注意事项
- 这里使用的是“最近一次缓存行命中”式简化缓存策略。
- 适合当前单任务、串行化处理流程。
